# 準備零件
* Arduino Pro Micro x 1
* 0.96吋OLED顯示螢幕 x 1
* 麵包板 x 1
* 麵包板線 x N
* 輕觸開關12*12*4.3MM x 2

* 有源蜂鳴器 x 1
